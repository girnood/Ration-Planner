# **App Name**: Ration Planner

## Core Features:

- Monthly Essentials List: Create and manage a list of monthly essential items with quantities and estimated costs.
- Spending Calculation: AI-powered calculation tool that calculates the estimated monthly food cost and offers a cost reduction plan. The tool determines if a list of requested essentials contains possible duplications, and opportunities for reduced expenses, without removing essential items. 
- Debt Tracking: Add and manage debts, track payment history, and view remaining balances.
- Profile and Budget: Set the application's overall spending target for the month.

## Style Guidelines:

- Primary color: Calm teal (#4DB6AC), evoking a sense of balance and resourcefulness.
- Background color: Soft beige (#F5F5DC), providing a warm, neutral backdrop.
- Accent color: Muted orange (#FF8A65), used for highlighting key actions and information.
- Headline font: 'Poppins' sans-serif font, for a modern, geometric, readable style.
- Body Font: 'PT Sans' sans-serif font, for readability
- Simple, clear icons representing different categories of essentials (food, household, etc.).
- Clean, intuitive layout with a focus on easy navigation and readability.
